# Flutter mobile sensors
This Flutter app attempts to fulfill the following native mobile functionalities:
- Capture and manage photos using the camera
- Integrate accelerometer to detect gestures (shake)
- Obtain user's GPS location
- Convert coordinates into addresses (geocoding)
- Configure and manage complex permissions
- Create interactive experiences with native features
